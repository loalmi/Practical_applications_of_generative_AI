# Финальный проект: Пилот анкеты через silicon sampling

**Трек:** A — исследовательский (homo silicus)  
**Тема:** Silicon sampling — использование LLM как синтетического респондента для пилотирования социологической анкеты


## Описание проекта

В проекте проверяется гипотеза: можно ли использовать LLM для быстрого и дешёвого пилотирования социологической анкеты. LLM «входит в роль» человека с заданным демографическим профилем (пол, возраст, образование, работа) и отвечает на вопросы анкеты. Синтетические ответы сравниваются с реальными данными РМЭЗ (Российский мониторинг экономического положения и здоровья населения НИУ-ВШЭ).

**Зачем это нужно:**  
Пилотирование анкеты через LLM позволяет:
- Найти двусмысленные вопросы до запуска в поле
- Оценить, как разные социальные группы будут отвечать
- Сэкономить время и деньги на пилотных исследованиях


## Краткие результаты

- **Синтетические ответы примерно соответствуют реальным распределениям**, но более поляризованы: LLM реже выбирает неопределённые варианты («И да, и нет» — 8% в синтетике vs 25% в РМЭЗ)
- **Валидатор (`field_validator`) улучшает качество данных:** средняя уверенность модели в ответах — 0.78
- **Найдены двусмысленные вопросы (Q1, Q2, Q4)**, которые требуют уточнения формулировок
- **Основное ограничение:** LLM никогда не выбирает крайние положительные варианты («Полностью удовлетворен» — 0% в синтетике vs 10.4% в РМЭЗ), что указывает на системное смещение модели

| Метрика | Значение |
|---------|----------|
| Респондентов (синтетических) | 50 |
| Ответов (всего) | 250 |
| Средняя уверенность модели | 0.78 |
| Совпадение с реальными данными (Q1) | ~70% |

## Как воспроизвести эксперимент

## Данные

Данные РМЭЗ можно скачать с официального сайта:
```
https://www.hse.ru/rlms/
```

Положите файл `r33iall_84.dta` в папку `input/` перед запуском.

### 1. Установка зависимостей
```
pip install -r requirements.txt
```

### 2. Настройка API

Скопируйте .env.example в .env и добавьте свой API-ключ:
```
cp .env.example .env
```
Пример содержимого .env:
```
LLM_BASE_URL=https://openrouter.ai/api/v1
LLM_AUTH_TOKEN=sk-or-v1-...ваш_ключ...
LLM_MODEL=deepseek-v4-flash
DATA_PATH=input/r33iall_84.dta
OUTPUT_DIR=output/
N_PERSONAS=50
N_SURVEY_QUESTIONS=5
```
### 3. Очистка данных
```
python src/clean_data.py
```
**Что делает:** загружает данные РМЭЗ, заменяет пропуски, фильтрует по качеству интервью, сохраняет очищенный файл в output/rlms_clean.csv.

### 4. Создание синтетических профилей
```
python src/create_profiles.py
```
**Что делает:** создаёт 50 демографических профилей на основе стратифицированной выборки из РМЭЗ (возраст × пол × образование). Сохраняет в output/profiles.json.

### 5. Запуск опроса (silicon sampling)
```
python src/survey.py
```
**Что делает:** для каждого профиля и каждого вопроса отправляет запрос к LLM с промптом «Ты — человек с профилем X. Ответь на вопрос». Сохраняет 250 ответов в output/survey_results_validated.json.

**Время выполнения:** ~10–15 минут (50 профилей × 5 вопросов).

### 6. Анализ и визуализация
```
python src/analyze_all.py
```
**Что делает:**
- Сравнивает ответы с валидатором и без
- Сравнивает синтетические ответы с реальными данными РМЭЗ
- Строит графики (сохраняет в output/plots/)
- Сохраняет таблицы (сохраняет в output/tables/)
